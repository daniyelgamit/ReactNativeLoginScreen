## How to use
- `git clone https://github.com/daniyelgamit/ReactNativeLoginScreen.git`
- `cd ReactNativeLoginScreen`
- `yarn install`
- `cd ios`
- `pod install` 
- `cd ..`
- `react-native run-ios` or `react-native run-android`
